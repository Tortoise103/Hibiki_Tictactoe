# Hibiki_Tictactoe

본 프로젝트는 버추얼 스트리머 '코가미 히비키'님에 대한 팬게임 프로젝트입니다.
개인 취미생활의 영역이며, 포트폴리오나 공식 서류 제출에서는 제외됨을 알립니다.

히비키의 앞머리를 들추고 틱택토를 하는 아주 짖궂은 게임입니다.  
이미지는 직접 그린 팬아트이며, 코드는 HTML/CSS/JS 기반으로 작성되었습니다. 별도 설치나 서버 없이 파일만 열면 바로 플레이할 수 있어요.

## 주요 기능
 
- 손그림 캐릭터 아트 + 눈 깜빡임 애니메이션
- 솔로 모드 (PVE, 미니맥스 AI — 최적수를 두는 상대와 대결)
- 2인용 모드 (PVP, 마우스 하나로 번갈아 플레이)
- 칸마다 다르게 그려진 O/X 마킹 (자연스러운 손그림 느낌)
- 배경음악 On/Off 및 볼륨 조절
- 한국어 / English / 日本語 / Italiano 4개 언어 지원
## 실행 방법
 
1. 이 저장소를 다운로드하거나 클론합니다.
2. `forehead-tictactoe.html`과 `bgm.mp3`가 **같은 폴더**에 있는지 확인합니다.
3. `forehead-tictactoe.html`을 더블클릭해서 브라우저로 엽니다.
4. 화면을 한 번 클릭하면 배경음악이 재생을 시작합니다. (브라우저 자동재생 정책 때문에 첫 클릭이 필요해요.)
```bash
git clone <이 저장소 주소>
cd <저장소 폴더>
# forehead-tictactoe.html 파일을 브라우저로 열기
```
 
## 커스터마이징
 
### 배경음악 교체하기
 
`bgm.mp3` 파일을 원하는 음원으로 교체하면 됩니다. **파일명은 반드시 `bgm.mp3`로 동일하게 유지**해주세요.
 
- 형식은 mp3를 권장합니다. (다른 포맷을 확장자만 mp3로 바꾸면 재생되지 않을 수 있어요.)
- 배포하거나 공유할 경우, 사용하는 음원의 라이선스를 꼭 확인해주세요.
### 캐릭터 아트 교체하기
 
`forehead-tictactoe.html` 안에 이미지가 base64로 내장되어 있습니다. 코드 내 `.face-img` 관련 이미지와 `MARK_IMAGES` 부분을 원하는 그림으로 교체하면 됩니다. 이마 격자 위치(`.forehead-grid`의 `left/top/width/height` 값)는 그림에 맞춰 다시 계산해야 합니다.
 
## 기술 스택
 
- Vanilla HTML / CSS / JavaScript (프레임워크 없음)
- 미니맥스 알고리즘 기반 AI
- HTML5 Audio API 

## 라이선스 및 크레딧
 
- 이 프로젝트는 **비공식 팬 메이드 콘텐츠**이며, 상업적 목적이 없습니다.
- 캐릭터 아트는 직접 제작했습니다.
- **이전 버전의 이미지에 디테일을 추가하고 약간의 수정을 가한 이미지가 포함된 버전입니다.**
